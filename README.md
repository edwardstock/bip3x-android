# bip3x-android
BIP39 library for android. Gradle package

## Usage:

```kotlin
allProjects {
    repositories {
        mavenCentral()
    }   
}

dependencies {
    implementation("com.edwardstock.android:bip3x:1.0.0")  
}
```

See [tests](bip3x/src/test/java/com/edwardstock/bip3x) for usage examples
